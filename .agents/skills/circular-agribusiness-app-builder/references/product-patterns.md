# Circular Agribusiness Product Patterns

Use these patterns when selecting or critiquing app concepts.

## Waste Stream Marketplace

Matches farms, food processors, markets, restaurants, haulers, composters, biochar producers, insect farms, digesters, and buyers.

- Core loop: list feedstock or product, verify quality, match buyer, coordinate pickup, record transaction.
- Data: location, volume, timing, moisture, contamination, price, quality grade, photos, pickup requirements.
- Risk: marketplaces fail without concentrated supply, trust, logistics, and repeated demand.

## Farm Input Advisory

Recommends compost, biochar, organic amendments, nutrient plans, and circular practices.

- Core loop: enter crop/soil/farm constraints, receive plan, track application, measure outcome.
- Data: soil tests, crop, acreage, amendment analysis, weather, yields, costs.
- Risk: recommendations need local agronomic validity and clear liability boundaries.

## MRV and Impact Accounting

Tracks waste diversion, emissions, soil carbon proxies, nutrient recovery, and circularity metrics.

- Core loop: capture activity data, calculate impact, attach evidence, generate report.
- Data: weights, distances, feedstock, processing method, energy use, application records, photos, receipts.
- Risk: weak methods, unverifiable claims, and carbon-credit complexity.

## Traceability and Quality System

Documents source, processing, test results, certifications, and chain of custody.

- Core loop: batch creation, input records, process steps, lab results, buyer documentation.
- Data: batch IDs, suppliers, dates, temperatures, test results, locations, certificates.
- Risk: too much manual entry unless the workflow is very focused.

## Circular Logistics Optimizer

Plans collection, routing, processing capacity, inventory, and delivery.

- Core loop: collect demand and supply, assign route, confirm pickup/delivery, update inventory.
- Data: addresses, vehicle capacity, pickup windows, volume, storage, processing capacity.
- Risk: high operational complexity and messy field conditions.

## Cooperative Operations Platform

Helps farmer groups coordinate inputs, shared equipment, training, compliance, and sales.

- Core loop: enroll members, plan activities, track contributions, allocate benefits, report results.
- Data: members, plots, activities, inventory, payments, documents.
- Risk: governance and incentives matter more than software polish.
